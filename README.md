# Secret Project

## Development

Make sure you have bun installed.

Then, `bun install`

Either use tauri dev mode `bun tauri dev` or web dev mode `bun dev` (recommended)

Open localhost:3000 in browser (chrome/ium recommended) to access the web dev build.

## Building

```
bun install
bun tauri build
```
Build is somewhere in src-tauri/target/release
